Bem-vindo ao Figma → Elementor

Veja em 1 minuto como funciona o fluxo completo antes de começar.

Fluxo geral em 4 passos

  • 1. Examinar o layout Figma (Linter) – corrigir auto layout, nomes e estrutura.
  • 2. Configurar chaves: Licença do plugin + chave de IA (opcional).
  • 3. Conectar ao seu WordPress – informar URL do site e testar conexão.
  • 4. Converter para Elementor – gerar JSON, menus, imagens e ícones.

Examinar layout e Linter

• Use a aba Validador / Análise para examinar o frame selecionado.
• O plugin identifica problemas: falta de Auto Layout, grupos desnecessários, textos soltos.
• Corrija o layout no Figma antes de compilar para melhores resultados.

Compilação com e sem IA

• Modo sem IA (No-AI) – usa regras fixas e heurísticas.
• Modo com IA – envia um snapshot para sua IA configurada.
• A IA ajuda a detectar padrões complexos (cards, grids, seções especiais).

Chave de IA do usuário

• O plugin não fornece créditos de IA: você conecta sua conta (OpenAI, Google Gemini).
• Na aba Configurações de IA, informe sua API key e modelo.
• Sem chave de IA, o plugin funciona apenas em modo No-AI.

Conectando ao WordPress

• Na aba Conexão WordPress, configure a URL do site (ex: https://seusite.com).
• Valide sua licença através do portal figmatoelementor.pljr.com.br.
• Uma vez conectado, você pode enviar JSON, imagens e menus diretamente.

Exportar imagens e ícones

• Ao compilar, o plugin exporta imagens e ícones marcados para export.
• Os arquivos são enviados otimizados para a Biblioteca Media do WordPress.
• No JSON gerado, os widgets de imagem já apontam para as URLs corretas.

Dica: Agrupar elementos em uma única imagem

• Para fundos complexos ou colagens, agrupe os elementos no Figma.
• Trave o grupo (lock) para indicar que deve ser exportado inteiro.
• Isso simplifica o HTML no Elementor e melhora a performance.
